# Dev-777-hub-roblox-script
um script para devs criarem seus scripts e funciona em qualquer executor!

# Como usar?

Quando você executar o script ira aparecer uma GUI com scripts para você programar seus scripts lua.

# Funções

Janela arrastavel.
Botão de fechar e minimizar.
Botão control direito para minimizar e mostrar.
Animações bonitas na Gui.

# Botões:

Dex Explorer DPP (decompilador de jogos no roblox local)
infinite yeld (script onde você consegue ter uma variedade de comandos de administrador local para desenvolver.)
Simple spy (script onde ele pega os remotes events e te da o codigo para fazer script FE ou que todos vejam.)
Cordenadas (um script onde ele te mostra as cordenadas do seu jogador e te deixa copiar elas.)
Lua executor (um executor de scripts lua ja integrado no jogo.)
